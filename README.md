# DevelopersHub Data Science & Analytics Internship

## Task 1: Iris Dataset Exploration
- Dataset: iris.csv
- Description: Exploratory Data Analysis with scatter plots, histograms, and box plots.
- Insight: Petal features are more useful than sepal features for species classification.

## Task 2: Credit Risk Prediction
- Dataset: loan_dataset_20000.csv
- Description: Logistic Regression model to predict loan approval.
- Accuracy: ~80% (example)
- Insight: Applicant income and credit history are most influential.

## Task 4: Insurance Claim Prediction
- Dataset: insurance.csv
- Description: Linear Regression model to predict medical charges.
- Metrics: MAE, RMSE
- Insight: Age, BMI, and smoking status are important factors.

## Task 5: Personal Loan Acceptance Prediction
- Dataset: bank.csv
- Description: Logistic Regression model to predict loan acceptance.
- Accuracy: ~90% (example)
- Insight: Age, job, marital status, and previous outcomes influence acceptance.
